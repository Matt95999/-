# 人工智能行业日报

> Hugging Face Transformers 发布 v5.6.0 系列更新，新增 OpenAI Privacy Filter 模型并修复 FP8 和 Flash Attention 问题；llama.cpp 推出 b8963 等版本，优化 Vulkan 和 CUDA 性能；Ollama 发布 v0.21.3 及 v0.22.0，新增 Nemotron 3 Omni 等模型；Ai2 推出 OlmoEarth 嵌入导出功能和科学发现智能体评估基准。

## 今日主线

### 产品与智能体

Hugging Face Transformers、llama.cpp 和 Ollama 均发布重要版本更新，涉及新模型支持、性能优化和 bug 修复。

#### Hugging Face Transformers 发布 v5.6.0 系列版本，新增 OpenAI Privacy Filter 模型

- Hugging Face Transformers 发布 v5.6.0 版本，新增 OpenAI Privacy Filter 模型，这是一个用于识别个人身份信息的双向 token 分类模型。
- 后续补丁 v5.6.1 修复了 Flash Attention 路径的 AttributeError 问题，v5.6.2 修复了 Qwen 3.5 和 3.6 MoE 模型在 FP8 下的运行问题。
- 此外，v5.7.0 版本新增 Laguna 模型。
- 结论：Transformers 库持续迭代，隐私过滤和模型兼容性得到增强。
- 影响：开发者可更安全地处理文本中的隐私信息，同时 FP8 和 Flash Attention 的修复提升了模型推理的稳定性和性能。
- 原文：https://github.com/huggingface/transformers/releases/tag/v5.6.0 | https://github.com/huggingface/transformers/releases/tag/v5.6.1 | https://github.com/huggingface/transformers/releases/tag/v5.6.2 | https://github.com/huggingface/transformers/releases/tag/v5.7.0

#### llama.cpp 发布 b8963 等版本，优化 Vulkan 和 CUDA 性能

- llama.cpp 发布 b8963 版本，在 Vulkan 后端合并了 Q4_K/Q5_K 的 scale 加载操作，提升推理效率。
- b8964 版本在推理完成后重新激活 reasoning budget，b8966 版本为 CUDA 后端添加了 flash-attn 支持，支持 DKQ=320/DV=256 和 ncols2=32 配置。
- 结论：llama.cpp 在 Vulkan 和 CUDA 上的性能持续优化，推理效率提升。
- 影响：使用 Vulkan 和 CUDA 后端的用户将获得更快的推理速度，尤其适合本地部署和边缘设备。
- 原文：https://github.com/ggml-org/llama.cpp/releases/tag/b8963 | https://github.com/ggml-org/llama.cpp/releases/tag/b8964 | https://github.com/ggml-org/llama.cpp/releases/tag/b8966

#### Ollama 发布 v0.21.3 和 v0.22.0，新增 Nemotron 3 Omni 等模型

- Ollama 发布 v0.21.3 版本，API 支持将 "max" 作为 think 值。
- v0.22.0 版本新增多个模型，包括 NVIDIA 的 Nemotron 3 Omni、Poolside 的首个开源编码模型等。
- v0.22.1 版本在 MLX 后端实现了跨多个序列的批量采样。
- 结论：Ollama 快速迭代，模型库和 API 功能持续扩展。
- 影响：用户可便捷地运行新模型，API 改进提升了推理控制的灵活性。
- 原文：https://github.com/ollama/ollama/releases/tag/v0.21.3 | https://github.com/ollama/ollama/releases/tag/v0.22.0 | https://github.com/ollama/ollama/releases/tag/v0.22.1-rc0

### 模型与推理

Ai2 推出 OlmoEarth 嵌入导出功能和科学发现智能体评估基准，推动地球观测和科学推理领域发展。

#### Ai2 推出 OlmoEarth 嵌入导出功能，支持自定义地球观测嵌入

- Ai2 在 OlmoEarth Studio 中推出嵌入导出功能，用户可从 OlmoEarth 基础模型导出自定义地球观测嵌入。
- 这些嵌入可用于相似性搜索、少样本映射和变化检测等下游任务。
- 结论：OlmoEarth 嵌入功能降低了地球观测领域 人工智能 应用的门槛。
- 影响：研究人员和开发者可更高效地利用遥感数据进行定制化分析，推动地理空间 人工智能 应用。
- 原文：https://allenai.org/blog/olmoearth-embeddings

#### Ai2 发布科学发现智能体评估基准，揭示当前 人工智能 的局限性

- Ai2 开发了 ScienceWorld 和 DiscoveryWorld 两个基准，用于评估 人工智能 科学智能体的能力。
- 评估结果显示，即使是强大的 人工智能 科学智能体，在解决人类科学家日常处理的问题时仍存在困难。
- 结论：当前 人工智能 在科学发现任务上远未达到人类水平。
- 影响：该基准为科学 人工智能 研究提供了标准化评估工具，有助于识别和改进智能体的推理短板。
- 原文：https://allenai.org/blog/evaluating-scientific-discovery-agents
