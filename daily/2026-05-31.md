### DeepSeek

DeepSeek API文档更新，支持V4-Pro和V4-Flash模型，同时将deepseek-coder与deepseek-chat合并升级为V2.5，兼容旧接口。

#### DeepSeek API 更新：支持 V4-Pro 和 V4-Flash 模型

- DeepSeek API 新增 V4-Pro 和 V4-Flash 模型，可通过 OpenAI ChatCompletions 和 Anthropic 接口访问。
- deepseek-chat 和 deepseek-reasoner 升级至 DeepSeek-V3.2，分别对应非思考模式和思考模式。
- 推出临时端点 DeepSeek-V3.2-Speciale，定价与 V3.2 相同，不支持工具调用。
- 结论：DeepSeek 通过多模型支持和接口兼容性，提升 API 灵活性和开发者体验。
- 影响：开发者可无缝切换模型，降低迁移成本，加速应用集成。
- 原文：https://api-docs.deepseek.com/updates#deepseek-v4 | https://api-docs.deepseek.com/updates#deepseek-v32 | https://api-docs.deepseek.com/updates#deepseek-v32-speciale

#### deepseek-coder 与 deepseek-chat 合并升级为 DeepSeek V2.5

- DeepSeek V2 Chat 和 DeepSeek Coder V2 模型合并升级为新模型 DeepSeek V2.5。
- API 用户可通过原有端点访问新模型，保持向后兼容。
- 结论：DeepSeek 通过模型合并简化产品线，同时保持兼容性。
- 影响：开发者无需修改代码即可获得更强模型能力，降低维护成本。
- 原文：https://api-docs.deepseek.com/updates#deepseek-coder--deepseek-chat-upgraded-to-deepseek-v25-model
