### Kimi

Kimi 发布长思考模型 API，K2 Think 模型及 turbo 版本正式上线，同时调整 API 价格与 TPM 配额。

#### Kimi 长思考模型 API 正式发布

- Moonshot 人工智能 开放平台正式推出 Kimi 长思考模型 API，支持长文本数据处理，并发布 K2 Thinking 模型及开源版本。
- 该模型在 智能体 和推理能力上全面提升，同时 API 价格调整，turbo 系列降价。
- 结论：Kimi 通过长思考模型 API 强化了在复杂推理和长文本场景的竞争力。
- 影响：开发者可调用更强大的推理模型，降低长文本应用成本，推动 智能体 类应用发展。
- 原文：https://platform.moonshot.cn/blog/posts/changelog | https://platform.moonshot.cn/blog/posts/k2-think | https://platform.moonshot.cn/blog/posts/k2-turbo-discount

#### Moonshot 人工智能 博客更新：K2 Think 模型及 turbo 版本发布

- Kimi K2 Think 模型及其 turbo 版本正式发布，同时提升 tier 等级默认 TPM 数值，turbo 系列模型降价。
- 更新还包括 CLI 支持文档、快速开始文档以及五折活动结束等细节。
- 结论：Kimi 通过模型迭代和价格调整，持续优化开发者体验和模型可用性。
- 影响：降价和 TPM 提升降低了开发者使用门槛，有助于扩大 Kimi 生态。
- 原文：https://platform.moonshot.cn/blog/posts/changelog#2025%E5%B9%B411%E6%9C%886%E6%97%A5

### Gemini

Gemini API 更新日志显示多模态搜索增强，Gemini 3.1 Flash Image 和 Pro Image 模型正式可用。

#### Gemini API 更新：多模态搜索增强与新模型 GA

- Gemini API 更新日志显示，文件搜索功能已更新以支持多格式搜索，gemini-embedding-2 模型可本地嵌入和搜索图像。
- Gemini 3.1 Flash Image 和 Gemini 3.1 Pro Image 模型正式发布（GA），同时 gemini-3.5-flash 也进入 GA 阶段。
- 结论：Google 通过多模态模型 GA 和搜索增强，巩固其在多模态 人工智能 领域的领先地位。
- 影响：开发者可集成更强大的图像理解和搜索能力，推动多模态应用落地。
- 原文：https://ai.google.dev/gemini-api/docs/changelog#05-28-2026

### DeepSeek

DeepSeek API 支持 V4-Pro 和 V4-Flash 模型，同时 deepseek-coder 和 deepseek-chat 升级至 V2.5。

#### DeepSeek API 支持 V4-Pro 和 V4-Flash 模型

- DeepSeek API 更新，新增 V4-Pro 和 V4-Flash 模型，可通过 OpenAI ChatCompletions 和 Anthropic 接口访问。
- 同时 deepseek-chat 和 deepseek-reasoner 升级至 V3.2，并推出临时端点 V3.2-Speciale。
- 结论：DeepSeek 通过快速迭代模型版本，保持 API 的先进性和兼容性。
- 影响：开发者获得更强大的模型选择，但需注意临时端点的有效期限制。
- 原文：https://api-docs.deepseek.com/updates#deepseek-v4

#### deepseek-coder 和 deepseek-chat 升级至 DeepSeek V2.5

- DeepSeek V2 Chat 和 DeepSeek Coder V2 模型合并升级为 DeepSeek V2.5，API 用户可通过旧名称访问新模型。
- 该升级旨在统一模型接口，提升编码和对话能力。
- 结论：模型合并简化了 API 调用，同时提升了整体性能。
- 影响：开发者无需修改代码即可获得更优模型，降低了迁移成本。
- 原文：https://api-docs.deepseek.com/updates#deepseek-coder--deepseek-chat-upgraded-to-deepseek-v25-model

### GLM

智谱发布 GLM-5.1 旗舰模型，编码能力大幅增强，支持长达 8 小时的自主任务闭环。

#### 智谱发布 GLM-5.1 旗舰模型

- 智谱 人工智能 发布 GLM-5.1 新一代旗舰模型，编码能力大幅增强，长程任务支持独立持续工作长达 8 小时。
- 模型在自主规划、持续执行、问题修复与策略迭代上展现更强工程智能，综合能力全面对齐国际前沿。
- 结论：GLM-5.1 在长程自主任务能力上取得突破，提升了国产模型的竞争力。
- 影响：企业用户可利用该模型实现复杂自动化工作流，减少人工干预。
- 原文：https://docs.bigmodel.cn/cn/update/new-releases#%E5%85%AC%E5%91%8A%E9%80%9A%E7%9F%A5
