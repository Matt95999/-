### Codex

本期 Codex 入选 3 条高置信线索，主线集中在 Codex app 26.616 的 Record & Replay 功能、ChatGPT for iOS 1.2026.160 的工作区文件浏览器，以及 Codex 功能在欧洲经济区、英国和瑞士的扩展。

#### Codex app 26.616 新增 Record & Replay 功能

- 事实：Codex app 26.616 版本为 macOS 用户新增 Record & Replay 功能，可将演示的工作流程录制为可复用的技能。该功能初始暂不支持欧洲经济区、英国和瑞士。
- 事实：同期版本 26.609 为 Plus 和 Pro 用户增加了速率限制重置银行，并推出推荐邀请机制。版本 26.608 则新增从 Claude Code 和 Claude Cowork 迁移至 Codex 的导入流程，并重新设计了插件屏幕。
- 结论：Codex 通过 Record & Replay 和竞品迁移功能，加速产品化并争夺开发者市场。
- 影响：降低自动化门槛，吸引非专业开发者；直接挖角 Claude 用户，加剧竞争。
- 原文：https://developers.openai.com/codex/changelog/#codex-app-26-616 | https://developers.openai.com/codex/changelog/#codex-app-26-609 | https://developers.openai.com/codex/changelog/#codex-app-26-608

#### ChatGPT for iOS 1.2026.160 新增工作区文件浏览器

- 事实：ChatGPT for iOS 1.2026.160 版本新增工作区文件浏览器，支持预览文件和将工作区路径链接到提示中。同时新增目录选择器，可在启动新线程时选择工作区文件夹。
- 事实：此前版本 1.2026.153 增加了分支选择、工作树创建和环境设置脚本支持；版本 1.2026.146 增加了可选的 Face ID 或密码锁，以及队列/转向默认行为设置。
- 结论：ChatGPT iOS 版通过集成 Codex 功能，将人工智能编程能力延伸至移动端。
- 影响：提升移动端开发效率，扩大 Codex 用户覆盖场景。
- 原文：https://developers.openai.com/codex/changelog/#chatgpt-for-ios-1-2026-160 | https://developers.openai.com/codex/changelog/#chatgpt-for-ios-1-2026-153 | https://developers.openai.com/codex/changelog/#chatgpt-for-ios-1-2026-146

### Claude Code

本期 Claude Code 入选 1 条高置信线索，重点在于 Claude Code 2.1.185 的 API 超时提示优化，以及此前版本中破坏性 git 命令的安全限制和 /config 快捷设置。

#### Claude Code 2.1.185 优化 API 超时提示并增强安全控制

- 事实：Claude Code 2.1.185 版本将流式传输停滞提示从“未收到 API 响应·正在重试…”改为“等待 API 响应·将在…后重试”，并将触发时间从静默 10 秒延长至 20 秒。
- 事实：此前版本 2.1.183 改进了自动模式安全性，阻止未明确要求时执行破坏性 git 命令（如 git reset --hard）。版本 2.1.181 新增 /config key=value 语法，支持在提示中直接设置任意配置项。
- 结论：Claude Code 通过超时优化和安全增强，提升开发者体验和企业级适用性。
- 影响：减少误报，提升效率；安全限制增强企业信心。
- 原文：https://code.claude.com/docs/en/changelog#2-1-185 | https://code.claude.com/docs/en/changelog#2-1-183 | https://code.claude.com/docs/en/changelog#2-1-181
