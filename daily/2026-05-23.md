### Gemini

今日 Gemini 发布版本说明，包括 Gemini 3.5 Flash 正式版和 Gemini 3.1 Flash-Lite 正式版，以及文件搜索支持多模态。

#### Gemini API 版本说明：3.5 Flash 和 3.1 Flash-Lite 正式版发布

- Google 发布 Gemini API 版本说明，宣布 Gemini 3.5 Flash 正式版（GA）上线，该模型在持续前沿性能上表现最佳。
- 同时发布 Gemini 3.1 Flash-Lite 正式版，针对速度、可扩展性和成本效益优化。
- 文件搜索功能更新，支持多模态搜索，可原生嵌入和搜索图像。
- 结论：Gemini 通过发布新模型和功能更新，持续强化其 API 产品线，覆盖高性能和轻量级需求。
- 影响：对开发者而言，提供了更多模型选择，兼顾性能与成本；对行业而言，Google 在模型迭代速度上保持领先。
- 原文：https://ai.google.dev/gemini-api/docs/changelog#05-19-2026 | https://ai.google.dev/gemini-api/docs/changelog#05-07-2026 | https://ai.google.dev/gemini-api/docs/changelog#05-05-2026

### DeepSeek

今日 DeepSeek 主线为 API 更新日志，支持 V4-Pro 和 V4-Flash 新模型，以及 deepseek-coder 和 deepseek-chat 升级至 V2.5 模型。

#### DeepSeek API 更新：支持 V4-Pro 和 V4-Flash 新模型

- DeepSeek API 更新日志显示，现已支持 V4-Pro 和 V4-Flash 模型，可通过 OpenAI ChatCompletions 接口和 Anthropic 接口访问。
- 同时，deepseek-chat 和 deepseek-reasoner 升级至 DeepSeek-V3.2 模型，分别对应非思考模式和思考模式。
- 还推出了临时端点 DeepSeek-V3.2-Speciale，定价与 V3.2 相同，但不支持工具调用。
- 结论：DeepSeek 通过快速迭代模型版本，持续提升 API 能力，并保持与主流接口的兼容性。
- 影响：对开发者而言，新模型提供了更强的性能和灵活性；对行业而言，DeepSeek 在开源和 API 服务上保持活跃。
- 原文：https://api-docs.deepseek.com/updates#deepseek-v4 | https://api-docs.deepseek.com/updates#deepseek-v32 | https://api-docs.deepseek.com/updates#deepseek-v32-speciale

#### deepseek-coder 和 deepseek-chat 升级至 DeepSeek V2.5 模型

- DeepSeek 将 DeepSeek V2 Chat 和 DeepSeek Coder V2 模型合并升级为 DeepSeek V2.5 模型。
- 为保持向后兼容，API 用户可通过原有端点访问新模型。
- 结论：DeepSeek 通过模型合并升级，简化了模型线，同时保持 API 兼容性。
- 影响：对开发者而言，升级后模型能力更强，且无需修改代码；对行业而言，模型合并趋势可能简化选择。
- 原文：https://api-docs.deepseek.com/updates#deepseek-coder--deepseek-chat-upgraded-to-deepseek-v25-model

- 该发布伴随 K2 Thinking 模型开源，全面提升 智能体 和推理能力，同时 K2 Turbo API 价格下调。
- Kimi 通过开放平台博客详细记录了新功能发布、模型更新和价格调整信息。
- 结论：Kimi 长思考模型 API 的发布标志着其在推理和 智能体 能力上的重要升级，并通过降价策略吸引开发者。
- 影响：对开发者而言，Kimi 提供了更强的长文本推理和 智能体 能力，可能推动更多复杂应用落地；对行业而言，加剧了模型推理能力的竞争。

### GLM

今日 GLM 发布 GLM-5.1 新一代旗舰模型，编码能力增强，支持长达 8 小时的自主长程任务。

#### 智谱 人工智能 发布 GLM-5.1 新一代旗舰模型

- 智谱 人工智能 发布 GLM-5.1 新一代旗舰模型，编码能力大幅增强，长程任务（Long Horizon Task）显著提升。
- 该模型支持一次任务中独立、持续工作长达 8 小时，实现从规划、执行到交付的完整闭环。
- 在自主规划、持续执行、问题修复与策略迭代上展现更强的工程智能，综合能力全面对齐国际前沿。
- 结论：GLM-5.1 的发布标志着智谱 人工智能 在长程自主任务和编码能力上的重大突破。
- 影响：对开发者而言，GLM-5.1 提供了强大的自主任务执行能力，可能改变复杂工程任务的开发模式；对行业而言，国产模型在长程任务上达到新高度。
- 原文：https://docs.bigmodel.cn/cn/update/new-releases#%E5%85%AC%E5%91%8A%E9%80%9A%E7%9F%A5
