DeepSeek API 支持 V4-Pro 和 V4-Flash 模型

DeepSeek API 更新，新增 V4-Pro 和 V4-Flash 模型，可通过 OpenAI ChatCompletions 和 Anthropic 接口访问。

DeepSeek V2 Chat 和 DeepSeek Coder V2 模型合并升级为 DeepSeek V2.5，API 用户可通过旧名称访问新模型。
