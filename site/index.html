Hugging Face Transformers 发布 v5.6.0，新增 OpenAI Privacy Filter 等模型

Hugging Face Transformers 发布 v5.6.0 版本，新增 OpenAI Privacy Filter 模型，这是一个用于个人身份信息识别的双向 token 分类模型。

该版本还包含其他新模型和功能改进，后续补丁版本 v5.6.1 和 v5.6.2 修复了 Qwen 3.5 和 3.6 MoE 在 FP8 下的问题。

此外，v5.7.0 和 v5.8.0 版本也已发布，分别新增 Laguna 和 DeepSeek-V4 模型。

结论：Hugging Face Transformers 持续快速迭代，积极整合最新模型并修复关键问题。

影响：开发者可以及时获得新模型支持和性能修复，提升模型部署的稳定性和效率。

https://github.com/huggingface/transformers/releases/tag/v5.6.0

SGLang 发布 v0.5.9，优化 LoRA 权重加载与计算重叠

SGLang 发布 v0.5.9 版本，亮点包括 LoRA 权重加载与推理计算重叠，将 TTFT（首 token 生成时间）降低约 78%。

后续 v0.5.10 版本默认启用分段 CUDA 图捕获，减少内存开销；v0.5.11 版本升级至 CUDA 13 和 PyTorch 2.11。

结论：SGLang 通过优化推理流水线和升级底层依赖，显著提升推理性能。

影响：使用 LoRA 微调模型的用户将获得更快的推理速度，降低延迟。

https://github.com/sgl-project/sglang/releases/tag/v0.5.9

llama.cpp 发布 b9031，改进后端按需加载

llama.cpp 发布 b9031 版本，主要改进是仅在需要时加载后端，减少不必要的资源占用。

后续版本 b9033 和 b9037 分别带来 macOS/iOS 更新和 Hexagon 处理优化。

结论：llama.cpp 持续优化资源管理和多平台支持。

影响：在资源受限设备上运行 llama.cpp 的用户将获得更好的内存效率和性能。

https://github.com/ggml-org/llama.cpp/releases/tag/b9031

Ollama 发布 v0.23.1，支持 Gemma 4 MTP 推测解码

Ollama 发布 v0.23.1 版本，为 MLX runner 添加 Gemma 4 MTP（多 token 处理）推测解码支持，在 Mac 上可带来超过 2 倍的速度提升。

此前 v0.23.0 版本已支持 Claude Desktop 与 Ollama Launch 集成。

结论：Ollama 通过支持 Gemma 4 MTP 显著提升 Mac 上的推理速度。

影响：Mac 用户运行 Gemma 4 模型时推理速度翻倍，提升本地 人工智能 体验。

https://github.com/ollama/ollama/releases/tag/v0.23.1

Ai2 发布开源机器人基础模型 MolmoAct 2，推动具身智能发展。

Ai2 发布 MolmoAct 2，开源机器人基础模型

Ai2 发布 MolmoAct 2，这是一个完全开源的机器人基础模型，具备更快的 3D 动作推理能力，并附带一个大型双臂操作数据集。

该模型旨在为真实世界机器人任务提供基础能力，推动具身智能研究。

结论：MolmoAct 2 作为开源机器人基础模型，降低了机器人研究门槛。

影响：机器人研究者和开发者可以利用该模型加速机器人任务开发，促进开源机器人生态。

https://allenai.org/blog/molmoact2
